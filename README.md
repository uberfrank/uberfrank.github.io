# François Richard - Portfolio

Site web professionnel statique basé sur mon profil LinkedIn.

## À propos

Développeur de profession, prophète de cœur #ECTO. Site portfolio personnel présentant mon expérience professionnelle, mes compétences et mon engagement communautaire.

## Caractéristiques

- Design moderne et responsive
- Interface utilisateur élégante avec animations subtiles
- Sections : À propos, Expérience, Formation, Compétences, Engagement bénévole, Contact
- Compatible mobile, tablette et desktop
- Performance optimisée

## Technologies utilisées

- HTML5
- CSS3 (avec variables CSS et animations)
- JavaScript vanilla
- Google Fonts (Inter)

## Structure du projet

```
uberfrank.github.io/
├── index.html              # Page principale
├── styles.css              # Styles CSS
├── script.js               # Interactions JavaScript
├── assets/
│   └── images/
│       └── profile.jpg     # Photo de profil
└── README.md              # Ce fichier
```

## Développement local

Pour visualiser le site localement :

1. Cloner le dépôt
2. Ouvrir `index.html` dans un navigateur web
3. Ou utiliser un serveur local comme `python -m http.server` ou `npx serve`
